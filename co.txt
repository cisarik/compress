TASK B — Add image predictor model module

Create:

* src/primesymbolicmdl/image_predictors.py
* tests/test_image_predictors.py

Implement a tiny deterministic set of grayscale predictors.

Define a small model representation, for example a dataclass:

* ImagePredictorModel
  fields:

  * name: str
  * params: dict[str, int]

Implement prediction function:

* predict_pixel(model, col, row, width, height, left, up, up_left) -> int

All predictions must be clamped to 0..255.

Required predictors:

1. zero

   * prediction = 0

2. left

   * prediction = left

3. up

   * prediction = up

4. avg_left_up

   * prediction = floor((left + up) / 2)

5. gradient

   * prediction = clamp(left + up - up_left, 0, 255)

6. x_ramp

   * prediction = floor(255 * col / max(1, width - 1))

7. y_ramp

   * prediction = floor(255 * row / max(1, height - 1))

8. diagonal_ramp

   * prediction = floor(255 * (col + row) / max(1, width + height - 2))

9. checker

   * parameter: block
   * prediction = 0 or 255 depending on ((col // block) + (row // block)) % 2
   * support candidate block sizes: 1, 2, 4, 8, 16

Implement:

* default_image_predictor_models() -> list[ImagePredictorModel]
* render_image_predictor(model) -> str
* image_predictor_model_bits(model) -> int
* image_predictor_parameter_bits(model) -> int

Costing:

* Keep model_bits and parameter_bits conservative and documented.
* checker(block=...) must pay parameter bits.

TASK C — Add image predictor codec/cost module

Create:

* src/primesymbolicmdl/image_predictor_branch.py
* tests/test_image_predictor_branch.py

Implement:

* estimate_image_predictor_cost(image, model) -> dict

Input image is the existing GrayImage object from image_datasets.py.

For each pixel in row-major order:

* compute left/up/up_left from previously decoded/original context
* compute predicted value
* residual = pixel - predicted

Cost model:

* raw_bits = width * height * 8
* header_bits = conservative fixed cost, for example 64
* model_bits = image_predictor_model_bits(model)
* parameter_bits = image_predictor_parameter_bits(model)
* residual_width:

  * if all residuals are exactly 0, residual_width = 0
  * otherwise use the minimal fixed signed width needed for min_residual..max_residual
* residual_bits = residual_width * pixel_count
* total_bits = header_bits + model_bits + parameter_bits + residual_bits
* saving_bits = raw_bits - total_bits
* ratio_vs_raw = total_bits / raw_bits, or 1.0 for empty image edge cases
* include min_residual, max_residual, residual_width, pixel_count, model string

Implement:

* encode_image_predictor_payload(image, model) -> dict

Payload must contain enough to decode exactly:

* image width
* image height
* model name and params
* residuals
* original_size or pixel_count metadata

Implement:

* decode_image_predictor_payload(payload) -> bytes

It must reconstruct the original pixel bytes exactly using only:

* model
* width/height
* residuals
* previously decoded pixels

Implement:

* roundtrip_image_predictor(image, model) -> bytes

It should return the decoded pixel bytes.
